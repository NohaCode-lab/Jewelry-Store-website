# 💎 Jewelry Store Website

A premium jewelry store website built with **React** and **Vite**, designed to showcase luxury jewelry collections through a modern, elegant, and fully responsive user interface.

The project highlights beautiful product presentations, smooth navigation, reusable React components, and a clean user experience across all devices.

---

## ✨ Features

- 🛍️ Elegant product showcase for:
  - Rings
  - Necklaces
  - Earrings
  - Bracelets

- 🎨 Modern and luxury-inspired UI design

- 📱 Fully responsive layout
  - Mobile
  - Tablet
  - Desktop

- ⚡ Fast performance powered by Vite

- 🧩 Component-based architecture using React

- 🖼️ High-quality product imagery

- 🧭 Smooth and intuitive navigation

- 📖 Well-structured website sections:
  - Hero Section
  - About Section
  - Products Section
  - Testimonials Section
  - Contact Section
  - Footer

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/NohaCode-lab/Jewelry-Store-website.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd Jewelry-Store-website
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start the Development Server

```bash
npm run dev
```

### 5️⃣ Open in Browser

Visit:

```text
http://localhost:5173
```
